## To Run Project in your Local PC follow the following steps:

### Clone My Github Repository from:
```
https://github.com/Joyontokarmakar/real-estate-template.git
```

### Project Installation

``` bash
$ npm install
```

or

``` bash
$ yarn
```

### Run Project

``` bash
$ npm run dev 
```

or 

``` bash
$ yarn dev
```
